# Diabetes Prediction (Kaggle Playground)

This project predicts whether an individual has diabetes using demographic and lifestyle data.

## Dataset
The dataset comes from the Kaggle Playground Series competition.

Download it here:
https://www.kaggle.com/competitions/playground-series-s5e12

Place the dataset files in:

data/raw/

## Methods
- Exploratory Data Analysis
- Feature Engineering
- Logistic Regression
- Random Forest

## How to Run

1. Clone the repository
2. Download the Kaggle dataset
3. Place `train.csv` and `test.csv` in `data/raw/`
4. Run the notebook
